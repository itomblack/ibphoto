# RasPi Node Camera Control



Using Barebones gulp.js project. Gives you local server, liveReload, Sass, & jshint.

### getting started

Install [node](http://nodejs.org/) & [gulp](http://gulpjs.com/)

`brew install node`, `npm install -g gulp`

Install packages

`npm i`

Run Gulp

`gulp`


### Pi

requires Node JS and Gphoto2 to be set up on Pi
